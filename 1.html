    공부할떄는 항상 통계를 기반으로 해서 무엇을 해야할 지 알고 하자. 기본이 쉬운것이 사실 가장 중요한 것일수 있으니 처음에 기반을 잘 닦아야 한다
    가야할 길이 멀다고 너무 의기소침해하지 말자.

    배운것들 list

    1. tag (가장 중요한 문법, 모를땐 사전적 의미 찾아보기)
    h1 ~ /h1 : 1~6까지 몇번쨰로 중요한 제목이다(글짜크기) + 띄우기
    br : new line = enter -> 여러번 쓰면 원하는 만큼 띄어쓸 수 있음
    P ~ /P : 줄바꿈 + 한 단락이라는 말 -> 정보 하나 더주는 것
    li ~ /li : list 목차 만들기 -> 부모테그 : ul ~ /ul(unordered list), ol~/ol(넘버링 필요할때 ordered list)
    table tr td : 표 만들기의 부모자식 테그

    h1 첫번째로 중요한 제목
    br 새로운 줄로 엔터

    2. 속성 attribute (테그의 심화된 문법)
    img : 이미지를 넣는 것 근데 이것만으로는 부족 img src=따옴표 안에 위치 넣으면 사진이 넣어짐(source의 줄임말이라고 보면 됨)
    테그의 의름만으로는 정보가 부족할 때, 속성을 통해 더 많은 의미 부여 가능


    3. 꿀팁들
    ctrl+o 하면 온라인에서 내 컴에 있는 페이지 열수 있음
    검색할떄는 html 00 tag 해서 하고 예제먼저 보고 definition 보기
    ctrl 누르고 커서 클릭해서 한번에 적으면 꿀띠 >ㅁ<
    html specification 쓰면 w3c에서 권고한 표준 있음!

    4. 구조 만드는 방법
    문장 -> 페이지 -> 책 : 많은 정보 잘 정리정돈 하는 책 = 구조
    head : 제목 타이틀과 메타는 이 본문의 정보가 이거고 이걸로 만들어져있다고 말해줌!!
    body : 본문 - html의 모든 테그는 바디테그나 헤드테그 중 하나에는 무조건 해당함
    html : 뭘까 - body와 head를 감싸는 최고위층 테그가 html!! only 1

    title : 웹페이지 제목 사용자에게 잘 알려줄 수 있음-> 책표지의 의미
    meta : 타이틀에 한글쓰면 안뜰때!컴퓨터는 0,1로 저장하는데 여러개 중에 utf-8 방식을 사용해서 적었으면 그걸로 읽으라고 해야함 meta charset=""해서 utf=8넣으면 그걸로 읽으라고 컴퓨터에게 말해준 것!

    링크거는 법
    the lords of HTML tags = a = anchor = 닻 = 정보의 바다에 정박한다 : 링크 link
